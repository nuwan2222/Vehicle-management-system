## 📌 Overview

A console-based Vehicle Service Center Management System built with **C++** and Object-Oriented Programming. Manages service appointments, vehicle records, customer accounts, and staff operations — all stored via text file handling with no external database required.

---

## 🧩 System Modules

### 👤 Customer Module
- Customer Registration & Login
- Forgot Password Recovery
- Schedule / Cancel Service Appointments
- View Service Records
- Update Personal Information

### 👨‍🔧 Staff Module
- Staff Registration & Login
- View Service Schedule
- Access & Search Vehicle Records
- View Vehicle Service History
- Mark Services as Completed

### 🚗 Vehicle Module
- Search Vehicle by Registration Number
- Display Vehicle Owner Details
- View Complete Service History
- Service Recommendations Tracking

---

## 💾 File Storage

| File | Purpose |
|---|---|
| `records.txt` | Customer accounts |
| `staff.txt` | Staff accounts |
| `appointment.txt` | Service appointments |
| `view.txt` | Customer information |

> No database setup required — all data persists through plain text files.

## 🚀 Getting Started

### Prerequisites
- Visual Studio (2019+) or CodeBlocks
- C++ compiler (MSVC / GCC)

### Run

```bash
# 1. Clone the repository
git clone https://github.com/nuwan2222/Vehicle-management-system.git

# 2. Navigate into the project
cd Vehicle-management-system
```

Then open the project in **Visual Studio** or **CodeBlocks**, compile all `.cpp` files, and run the executable. The menu-driven console will launch automatically.
